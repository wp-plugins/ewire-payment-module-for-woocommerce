== Description ==

Ewire Payment Module for WooCoommerce makes it possible to accept payment from scandinavia.
It is a simple setup. By adding this module to your setup you only need to create a Ewire Business
account at http://www.ewire.dk/create. Afterwards you are ready to receive payments.

== Installation ==

Installation of the module


1. Upload folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill out the needed values in the WooCommerce Payment Gateways page


== Frequently Asked Questions ==

= Where do I find CompanyID and/or Encryptionkey =

By logging into your Ewire account and choosing Business from the main menu, you will find those information

= The payment window returns an Ewire error =

Have you checked that the payment domain is added to your Ewire account. This is controlled by logging into your Ewire account and choosing the tab Business.
